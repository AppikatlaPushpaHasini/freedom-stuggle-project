# 🇮🇳 Freedom and Struggle

An interactive web-based project that showcases the contributions of Indian freedom fighters through a state-wise exploration of the Indian map.

## 📖 About the Project

Freedom and Struggle is an educational web application built using HTML, CSS, and JavaScript. The project aims to help users learn about India's freedom movement in an engaging and interactive way.

The application displays an interactive map of India. When users hover over a state, a popup appears showing brief information about notable freedom fighters from that region. Clicking on a state opens a dedicated page containing detailed information about the selected freedom fighter, including their achievements and contributions to India's independence.

## ✨ Features

- Interactive India map
- State-wise freedom fighter information
- Hover popups with short descriptions
- Dedicated pages for detailed biographies
- Information about achievements and contributions
- Responsive and user-friendly interface
- Pure front-end implementation using HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
